# TeamDraw
*Online multiplayer drawing game*

<p>
TeamDraw is a online multiplayer drawing game inspired by a game I played as a kid.
</p>
<p align="center">
<img width="75%" height="75%" src="https://firebasestorage.googleapis.com/v0/b/creatoratnight-babdb.appspot.com/o/images%2F68747470733a2f2f63726561746f7261746e696768742e636f6d2f6769746875622f746430312e706e67.png?alt=media&token=0acd3f04-9541-4bab-92ed-a633c855927b">
</p>
<p>
Each game a new canvas gets split up between 8 players, each player will only see their part of the canvas. 
The players get a theme and some guide lines so their drawings match up with those of the other players.
</p>
<p align="center">
<img width="75%" height="75%" src="https://firebasestorage.googleapis.com/v0/b/creatoratnight-babdb.appspot.com/o/images%2Fteamdraw_00.png?alt=media&token=f4709d25-a29c-4a52-a578-909449768620">
</p>
<p>
When all the players have finished their drawing, the whole canvas will be revealed.
</p>
